展示行列数据

## 如何使用

指定表格的数据源 `dataSource` 为一个数组
```javascript
const dataSource = [
  ['ezio', 18],
  ['mary', 20]
];

const columns = [
  { title: 'name', dataIndex: 0, id: '1' },
  { title: 'age', dataIndex: 1, id: '2' }
];

// or

const dataSource1 = [
  {name: 'ezio', age: 18},
  {name: 'mary', age: 16}
]

const columns1 = [
  { title: 'name', dataKey: 'name', id: '1' },
  { title: 'age', dataKey: 'age', id: '2' }
]
```

## API

### Table

| 属性 | 说明 | 类型 | 是否必须 | 默认值 |
| --- | ---- | --- | ------ | ------ |
| dataSource | 数据数组 | object[] | 是 | - |
| columns | 表格列的配置描述，具体项见下表 | ColumnItem[] | 是 | - |
| wrapperClassName | 自定义 table wrapper 的 className | string | 否 | - |
| styles | 自定义 table wrapper 的 style | React.CSSProperties | 否 | - |
| size | 设置 table 的大小类型 | mini &#124; middle &#124; large | 否 | mini |
| fixedHeader | 固定表头 | boolean | 否 | false |
| showIndex | 显示序号 | boolean | 否 | false |
| fixedSize | 固定前 x 列 | number | 否 | - |
| customGroupHeader | 自定义渲染 group title | (data, originNode) => React.ReactNode; | 否 | - |
| colSummaryData | 列总结数据 | object[] | 否 | - |
| pagination | 分页器 | 参考 [pagination](/components/pagination) 文档，为空则不显示分页 | 否 | - |
| rowSummaryCol | 行总结数据列，具体项见下表 | RowSummaryCol[] | 否 | - |
| zebraStripe | 斑马纹模式 | boolean | 否 | - |
| bordered | 边框模式 | boolean | 否 | - |

### Column
| 属性 | 说明 | 类型 | 是否必须 | 默认值 |
| --- | ---- | --- | ------ | ------ |
| title | 表格标题 | string | 是 | - |
| align | 列对齐状态 | left &#124; center &#124; right | 否 | left |
| dataIndex | 列数据在数据项中对应的路径，支持通过数组查询嵌套路径 `将在后续版本废弃` | number | 否 | - |
| dataKey | 列数据在数据项中对应的路径，支持通过数组查询嵌套路径 `推荐使用` | string | 否 | - |
| id | React 需要的 key，如果已经设置 dataKey 可以忽略 | string | 否 | - |
| defaultSortOrder | 默认排序方向 | asc &#124; desc | 否 | - |
| width | 列宽度 | string | 否 | - |
| render | 自定义单元格 | (data, originNode) => React.ReactNode | 否 | - |
| sorter | 自定义排序函数 | (a, b) => any | 否 | - |
| children | 用于创建嵌套分组表头 | ColumnItem[] | 否 | - |
| mergeRow | 开启合并单元格 | boolean | 否 | - |

### RowSummaryCol `继承 Column`
| 属性 | 说明 | 类型 | 是否必须 | 默认值 |
| --- | ---- | --- | ------ | ------ |
| totalData | 当同时开启行列总结时，用于取行列总计数据 | any | 否 | - |
