## API

| 属性 | 说明 | 类型 | 是否必须 | 默认值 |
| --- | ---- | --- | ------ | ------ |
| routeConfig | 路由配置文件 | <T = RouteProps> | 是 | RouteProps |
| exception | 异常渲染节点 | React.ComponentType | 否 | - |
| nodeContainer | 节点包裹容器 | (data: { component, route }, children) => React.ReactNode | 否 | - |

## RouteProps

| 属性 | 说明 | 类型 | 是否必须 | 默认值 |
| --- | ---- | --- | ------ | ------ |
| path | 路由路径 | string | 是 | - |
| component | 组件 | React.component | 是 | - |
| routes | 子节点 | RouteProps | 否 | - |
| redirect | 重定向 | string | 否 | - |
| strict | 严格匹配 | string | 否 | - |
| exact | 精确匹配 | string | 否 | - |
| key | 标识 | string | 否 | - |
| extraInfo | 额外信息 | any | 否 | - |
